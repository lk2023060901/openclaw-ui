# 前端技术栈

本项目的前端技术栈固定如下，不可变动：

- React
- Next.js
- Tailwind CSS

## 技术栈约束

前端实现必须基于以下组合完成：

- UI 框架：React
- 应用框架：Next.js
- 样式方案：Tailwind CSS

## 不可变更说明

以下内容视为项目级约束，不允许替换、混用或迁移到其它方案：

- 不替换为 Vue、Svelte、Angular 等其它前端框架
- 不替换为 Remix、Nuxt、Astro 等其它应用框架
- 不替换为 CSS Modules、Styled Components、Emotion、Ant Design 内建样式体系等其它主样式方案
- 不引入与 Tailwind CSS 相冲突的另一套主导型 UI 样式体系作为基础方案

## 实施要求

- 页面与组件开发统一使用 React 组件模型
- 路由、页面组织、构建与运行能力统一基于 Next.js
- 页面样式、布局、主题表达、响应式规则统一基于 Tailwind CSS

## 结论

本项目的前端技术栈为：React + Next.js + Tailwind CSS。

这是固定技术栈，后续前端实现不得变动。
